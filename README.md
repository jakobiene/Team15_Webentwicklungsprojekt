## Projekt
Webshop als Fullstack-Anwendung mit React (Frontend), Node.js/Express (Backend) und MySQL.


## Setup
### 1. Repository klonen
git clone <repo-url>
cd team15_webshop

### 2. Frontend starten
cd frontend
npm install
npm run dev

### 3. Backend starten
cd backend
npm install
npm run dev
